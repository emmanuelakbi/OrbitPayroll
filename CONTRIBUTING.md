# Contributing to OrbitPayroll

Thank you for your interest in contributing to OrbitPayroll!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/OrbitPayroll.git`
3. Create a branch: `git checkout -b feature/your-feature`
4. Make your changes
5. Run tests: `npm run test`
6. Commit: `git commit -m "feat: add your feature"`
7. Push: `git push origin feature/your-feature`
8. Open a Pull Request

## Development Setup

See [README.md](README.md) for detailed setup instructions.

## Code Style

- TypeScript for all code
- ESLint + Prettier for formatting
- Conventional commits (feat:, fix:, docs:, etc.)

## Testing

- Write tests for new features
- Ensure all tests pass before submitting PR
- Use property-based testing for critical logic

## Security

- Never commit secrets or API keys
- Report vulnerabilities privately (see [SECURITY.md](SECURITY.md))
